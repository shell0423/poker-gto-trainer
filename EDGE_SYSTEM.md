# 🏆 EDGE POKER 優勝システム（チートシート）

EDGE POKER のトーナメント構造に最適化した、**スタック深さベースの意思決定システム**。
トレーナー（`index.html`）の **構成「EDGE Push/Fold (Nash, +1bb ante)」** と
**クイズの出題範囲「EDGE POKER」** をこの文書とセットで使う。

---

## 0. EDGE の構造（前提）

- **12人開始 → 4人卓(4-max)で進行 → 6人になったら決勝（単卓6人→HUまで）**
- **初期8000=40bb / 1bb BBアンティ / 超ターボ**（3分ごとにブラインド上昇、15分時点で約8bb）
- ポイントは上位6位、チャンピオンシップは**優勝1名** → 基本は **chipEV（攻撃的）**

**最重要の事実：構造が速いので、勝敗の大半は「短いスタックの push/fold」で決まる。**
よって優勝システム＝**ジャム/コールのレンジを体に入れること**。

---

## 1. ステージ別の指針

### ステージ1：40〜25bb（4-max・開始〜数分）
- ポジションで開く。**UTGはタイト（後ろ3人）、BTN/SBは広く**スチール（アンティ分さらに広く）。
- OOPで膨らませない。マージナルは **3bet-or-fold**、ディープな postflop を避ける。
- 使う構成：**「EDGE 4-max 40bb +ante」**（序盤レンジ）。

### ステージ2：25〜15bb（4-max）
- **RFI-or-fold**。オープンへは**3bet-jam**（コールして postflop より、被せて降ろす/全in）。
- BTN/SB からは**オープンジャム**も増やす。
- 使う構成：EDGE Push/Fold の **15〜20bb** 帯。

### ステージ3：≤15bb（4-max → 決勝卓）＝本丸
- **純 push/fold**。first-in は Nash 通りにジャム、リジャム、相手のジャムには call チャート通り。
- 使う構成：**「EDGE Push/Fold」4-max → 6-max → HU**（スタック 8〜15bb）。

### 決勝卓（6人→HU・全員短い）
- 6-max → 3-max → HU と人数が減るほど**レンジは広がる**（後ろが減る＋アンティ）。
- 使う構成：EDGE Push/Fold の **6-max / HU**。

---

## 2. アンティと卓人数の効き方（暗記）

- **1bb BBアンティ＝毎ポットに死に金1bb** → ジャムもコールも**広くなる**（降ろせば得、コールも安い）。
- **卓が小さいほど（4-max・HU）レンジは広い**（後ろの人数が少ない）。
- だから 6-max の感覚より **1〜2段広く** 打つのが EDGE 最適。

### 目安（自前Nash, +ante, jam%）
| 卓 | ポジション | 10bb | 15bb |
|---|---|---|---|
| 4-max | UTG | ~41% | ~32% |
| 4-max | **BTN** | **~48%** | ~40% |
| 6-max | BTN | ~48% | ~42% |
| **HU** | **SB** | **~71%** | ~65% |

（正確な169ハンド表とコール域・EVはトレーナーの「EDGE Push/Fold」構成に全部入っている）

## 3. 練習法（トレーナー）

1. クイズタブ → **出題範囲＝「EDGE POKER」** にする（EDGEの局面だけ出る）。
2. **戦略クイズ**で jam/fold・call/fold を当てる。回答後に**理由とEV**が出る。
3. 間違えたら **復習モード** で間違いだけ繰り返す。
4. チャートタブで **「EDGE Push/Fold」** を開き、各スタックの色分け表を眺めて境界ハンドを覚える。

**ゴール：4-max と HU の push/fold を、考えずに出せるようにする。** これがEDGE優勝の土台。

---

## 4. 注意（精度の前提）

- EDGE Push/Fold は **chipEV の本物の Nash**（自前ソルバー＋1bbアンティ）。ICM は未モデル化。
- **優勝狙い（chipEV）にはこれで最適**。ただし「上位6位のポイント当落線」だけは、
  生き残り価値が上がるので **ほんの少しタイトに**寄せてよい（バブルだけ）。
- 序盤の 4-max 40bb レンジは**近似（学習用）**。push/fold は厳密。
